Invocation

Slash Command

/deliberate "topic" — starts a deliberation with your configured default panel

/deliberate --models opus,glm-5.1 "topic" — pick specific models

/deliberate --rounds 3 "topic" — set max rounds (default: 5)

/deliberate --duo "topic" — quick 2-model mode with Opus + your secondary

Mid-conversation

You can also start a deliberation mid-flow:

"Hey, I'm stuck on how to structure this auth middleware. Can you deliberate on it with GLM?"

Void detects the intent and launches a deliberation with current conversation context passed to both models.

Architecture

Deliberation Engine

Round-based conversation:

  1. All models receive the topic + codebase context
  2. Model A responds first (randomly selected or highest-tier)
  3. Model B receives A's response, responds with agreement/challenge/build
  4. Model C (if trio) receives both, synthesizes + adds
  5. Model A gets all feedback, revises — this is one "round"
  6. Repeat for N rounds or until convergence

Convergence detection: If all models agree on the approach for 2 consecutive rounds, auto-stop early.

Human injection: Press Enter at any point to inject your own thoughts into the next round. The models treat your input as the highest-priority signal.

Key Design Decision: Anti-Sycophancy Prompting

The system prompt for each model in the deliberation explicitly instructs them to challenge, not agree. Each model is told:

You are in a deliberation with other AI models. Your job is to make the
final answer BETTER, not to be polite. Specifically:
- If you see a flaw, call it out directly
- If you have a better approach, present it with evidence
- If you agree, say WHY and ADD something new — don't just echo
- If the previous model's revision addressed your concern, acknowledge
  it and move on to the next weakness
- Converge when the solution is genuinely strong, not to end the debate
