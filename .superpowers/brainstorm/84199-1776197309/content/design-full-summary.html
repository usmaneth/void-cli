Void CLI — Full Feature Design Summary

5 features that turn Void into a multi-model AI development platform

1. Fix Plugin Marketplace

Change 3 files to point void-plugins-official → claude-plugins-official. Unlocks all 33 official plugins (superpowers, figma, slack, etc.) for Void.

Scope: ~6 lines changed. Ships immediately.

2. OpenRouter Model Browser

Full-catalog model picker with live search, provider filter tabs (Tab to cycle), favorites (⌘+F to pin), smart context-aware suggestions, pricing display, and 1-hour model cache from OpenRouter API.

New files: openrouterModels.ts, modelSuggestions.ts. Modified: ModelPicker.tsx, modelOptions.ts, settings types.

3. Deliberation Room + Swarm Mode

/deliberate — 2-3 models in round-based debate. Anti-sycophancy prompts force real challenge. Human can inject thoughts mid-round. Auto-stops on convergence.

/swarm — Multi-model parallel implementation. Opus coordinates, decomposes task into workstreams, each model builds in isolated worktrees, coordinator merges + wires. Default assignments: Gemini → frontend, GPT-5.4 → backend, Opus → architecture.

New modules: src/deliberation/, src/swarm/. Registered as built-in skills with smart triggers.

4. Designer Agent (Gemini 3.1 Pro)

Built-in frontend design specialist. Two modes: /design for design-first visual specs, or auto-invoked mid-flow when visual work is detected. Reads existing design system (tailwind, tokens, components) before writing. Outputs production-grade, beautiful, modern UI code. Claude handles types/state/backend after.

Designer prompt emphasis:

  • Beautiful, modern UI components — not generic/template-looking
  • Smooth animations and micro-interactions on everything
  • Contemporary design trends — glassmorphism, gradients, depth
  • Premium feel — the kind of UI that makes people say "who designed this?"
  • Component library quality — every element polished, consistent, reusable
  • Pixel-perfect spacing, intentional typography, deliberate color choices

New module: src/agents/designer/. Direct Gemini API via @google/genai SDK.

5. Provider Auth (OpenAI + Google Gemini)

/provider add openai and /provider add gemini with macOS Keychain storage. Both also support env var fallback (OPENAI_API_KEY, GEMINI_API_KEY). Extends existing /provider command. Enables direct API calls for swarm workers and designer agent.

New: src/providers/openai.ts, src/providers/gemini.ts. Modified: provider command, settings types.

6. Smart Mode Triggers

All three modes (deliberate, swarm, designer) registered as built-in skills with whenToUse heuristics. The main Void agent dynamically suggests the right mode based on conversation context — never auto-launches, always asks first. Feels like a smart teammate saying "hey, this might be a good spot to bring in the team."
