Architecture

Data Flow

1. Fetch models — GET openrouter.ai/api/v1/models on first /model open. Cache for 1 hour in ~/.void/cache/openrouter-models.json.

2. Favorites — Stored in settings.json as favoriteModels: string[]. Toggle with ⌘+F hotkey while model is highlighted.

3. Smart suggestions — Lightweight heuristic based on recent tool use and file types in context. Sees .tsx files → suggests design-capable models. Sees test files → suggests reasoning models.

4. Provider filter — Tab cycles through providers extracted from model IDs (part before /). Filters the list in real-time.
